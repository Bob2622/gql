## CLI

```
$ npm u -g gql
$ gql -h

  Usage: gql [options] [command]


  Commands:

    merge [options] <glob>   Merge multiple GraphQL files into one
    format [options] <glob>  Formats GraphQL files

  Options:

    -h, --help     output usage information
    -V, --version  output the version number

```
